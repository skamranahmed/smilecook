## SmileCook
Recipe Sharing Platform

### Tech Stack

**Backend:** Golang

**Database:** MongoDB

**Deployment:** Docker,AWS ECR, AWS EC2

**Monitoring:** Prometheus, Grafana

**CI/CD:** CircleCI

**Logs:** ELK Stack
